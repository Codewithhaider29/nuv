# Nuvé Skincare Analysis App

Nuvé is an AI-powered personalized skincare analysis tool that provides detailed insights and custom routines for your healthiest skin.

## Features
- **Intelligent Analysis:** Scans and analyzes your skin's unique needs.
- **Mindful Beauty:** Focuses on realistic, holistic improvements.
- **Real-Time 3D Orb:** A visual "living model" of your skin's health.
- **AI Guidance:** Intelligent suggestions tailored specifically for you.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- GSAP (for scroll and reveal animations)
- Three.js / React Three Fiber (for 3D models)
- TanStack Router

## Getting Started

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`.
